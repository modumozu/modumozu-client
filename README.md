# 모두모주

공모주 청약 및 계좌 개설 일정을 알려주어 공모주 청약 접근성을 높이는 서비스

증권사 계좌 개설 20일 제한 때문에 공모주 청약을 놓치는 문제를 해결

### 핵심기능

- 보유 계좌 정보를 추가하면  공모주 일정 제공
- 계좌 개설로 인해 다른 공모주 신청에 제약이 생기는 경우 안내
- 개설 제한을 해제할 수 있는 팁 제공
